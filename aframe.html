Make a VR world in 10 lines of HTML code!

A-Frame is a free, open-source framework for making Virtual Reality on the Web using HTML and JavaScript.

You can use components to modify, extend, or change your design needs. You shouldn’t need to write JavaScript every time you want to build a new component. By learning A-Frame components, you can quickly start to build out interfaces using composable building blocks while writing little to no JavaScript.

Download v0.5.0 aframe.min.js

Let’s build our first world together …

<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>My First WebVR Scene • Made with A-Frame</title>
    <script src="https://aframe.io/releases/0.5.0/aframe.min.js"></script>
  </head>
  <body>
    <a-scene>
      <a-box position="-1 0.75 -3" rotation="0 45 0"
             color="#4CC3D9"></a-box>

      <a-sphere position="0 1.5 -5" radius="1.25"
                color="#EF2D5E"></a-sphere>

      <a-cylinder position="1 1 -3" radius="0.5" height="1.5"
                  color="#FFC65D"></a-cylinder>

      <a-plane position="0 0.25 -4" rotation="-90 0 0" width="4"
               height="4" color="#7BC8A4"></a-plane>

      <a-entity geometry="primitive: circle; radius: 40"
                rotation="-90 0 0"
                material="src: grid.png; repeat: 160 160"></a-entity>

      <a-sky src="sky.png"></a-sky>
    </a-scene>
  </body>
</html>
