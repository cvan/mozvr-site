Defining Virtual Reality on the Web

We help design the WebVR API standard, for displaying VR web pages in Firefox, our desktop browser, and Servo, a next-generation, parallel browser engine.

We make A-Frame, a Web framework for creating VR.

WebVR API

WebVR is an open collaboration between multiple stakeholders and is actively being developed as part of a W3C community group.

Developers can use WebVR today to create VR experiences in Firefox, Edge, Samsung Internet, Oculus, and Chrome for Android.

You can view ongoing work on the WebVR GitHub site, including an extension to the initial specification to improve head motion tracking and tools for creating a trusted UI environment.

A-Frame

Join the thriving community of A-Frame developers building VR experiences using simple HTML. Anyone can create fully interactive 3D worlds with very little experience, so there’s no need to learn linear algebra or code in C++.

A-Frame is built on the Three.js JavaScript 3D engine library. Want another option? PlayCanvas also has good documentation and an active community.
